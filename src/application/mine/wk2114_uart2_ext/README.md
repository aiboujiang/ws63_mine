# WK2114 UART2 扩展模块说明

本模块用于在 `mine` 目录下提供一个独立的 WK2114 接入骨架：

1. 使用 `UART2` 作为 WK2114 主接口（MRX/MTX）与芯片通信。
2. 提供子串口（1~4）波特率配置与 FIFO 发送的基础接口。
3. 通过 OLED 实时输出调试信息（状态、收发计数、数据预览）。

## OLED 页面布局

1. 第 1 行：标题（`WK2114 UART2 EXT`）。
2. 第 2 行：状态（例如 `STATE:SUB1 READY`）。
3. 第 3~5 行：最近一条数据预览（可显示 RX/TX 命令帧）。
4. 第 6 行：当前通道与波特率（`CH:U1 B:115200`）。
5. 第 7 行：RX 统计（最近长度 + 累计次数）。
6. 第 8 行：TX 统计（最近长度 + 累计次数）。

## 构建开关

在 `menuconfig` 中启用：

`Application -> Mine -> Support Mine UART2 Expand via WK2114.`

## 当前实现边界

1. 已实现：按规格书 UART 主接口协议发送寄存器写命令、FIFO 写命令。
2. 已实现：子串口基础配置流程（SPAGE/BAUD/PRES/SCR/FCR）。
3. 预留扩展：全量读回校验、全局中断/IRQ 处理、上层业务路由。
